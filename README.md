# 综述

游戏名称：蚀岛。
单人。
游戏为H5页面的游戏，使用ts+vue3制作。
主要面向手机用户，画面、操作主要考虑手机用户的使用、操作方便。

# 玩法

主要通过文字交互，玩家通过点击对应选项操作主角的动作、对话及行动。保证主角生成、成长、解密、完成任务、最终目标是从荒岛逃生（或适应）。

## 显示

显示分为过场画面、常态画面、战斗画面、菜单界面等。

### 过场画面

过场画面应根据具体场景进行单独设置，可配以背景图或浮动图的形式。没有多余的UI，画面中间显示文字，玩家点击任意位置继续显示下一段。（根据需要可取消替换显示，或继续显示在上一段下方）如果有需要玩家选择的地方，则显示对应按钮。

### 常态画面

画面最上方显示玩家的状态、日期和时间。
画面中间为主要显示区域，显示文字，例如当前环境描述，其他人的对话等，某些场景描述或人物描述中，玩家可通过此处互动，可互动内容应明显看出，例如蓝色下划线等。例如查看人物、查看环境中的某个值得查看的物体等。中间左侧或上方可以显示当前环境的图片。
画面下方为玩家的主要操作区域。分两种情况，第一种情况是常态无交互进行中时，左侧是系统、背包、属性、状态等操作按钮，右侧是当前场景特有的交互按钮，例如探索、劈柴、建造、与某人对话、进行战斗等。第二种情况是进入交互后特有的交互界面，例如进入对话时，此处显示玩家可选的回复内容，进入建造时显示可选的建造物等，根据实际情况一般得有返回离开等选项。

### 菜单界面

菜单界面是配置界面，可分为：
系统菜单界面：例如保存、读取、成就、返回主菜单等逻辑操作；
背包/装备界面：显示玩家当前的背包中东西和装备等；
状态/属性界面：显示玩家的属性、状态、技能等（可与上面合为一个）；

### 战斗画面

战斗画面渲染要与普通画面有分别差别，以防玩家误操作。战斗画面中的按钮应与战后结算按钮错开，以防玩家点击过快，错过结算内容。

画面上方显示玩家状态，玩家状态下面显示敌人状态和信息。画面中间显示战斗描述。画面下方是操作界面。例如攻击、逃跑等。
战斗结束后进入结算界面，显示战利品、描述等，玩家确认后方可回到常态画面，或常态画面（交互中）。

## 属性

玩家属性包括：生存属性、基础属性。
生存属性包括：
生命值：扣为0后死亡，游戏结束，随时间缓慢恢复，或使用道具回复，睡觉时加快回复。回复速度与体质相关。
饱食度：吃食物回复，为0时开始扣HP。
口渴度：同饱食度。
SAN：玩家的理智值，随着降低，世界也随之变化，具体到游戏中包括但不限于UI的变化、同样地点、物品描述的变化。某些选项也仅在某些SAN值范围内可见。低到一定程度时，玩家可能会出现不可控制的行为。遇到恐怖之事、学习无法理解知识、吃难以描述食物等会降低，睡觉、吃好食物、某些正向活动可以回复。智力越高越下降越容易，同时恢复也更多。
温暖度：根据玩家衣物与天气环境决定，不在适宜范围内有debuff，差太多会扣血，长期会影响体质。在适宜范围内有buff。

基础属性包括：
力量：力气大小，决定负重和是否能进行某些操作，相关运动可以提升。
敏捷：速度快慢，决定移动速度、战斗先后、是否可进行某些操作等，相关动作可以提升。
体质：体质优劣，规律作息、吃饭喝水可以提升，熬夜、长期处于异常状态下降。
智力：阅读、研究等可以提升，影响研究速度、是否可建设某些东西等。

## 技能

除属性之外，技能也影响玩家交互的效率或条件。
包括但不限于：侦察（更容易发现敌人、新地点、不容易被伏击、线索等）、砍伐、制作、攀岩、射击、斗殴、闪避、跑步、潜行等。同一个目的可以让玩家依赖于不同技能或属性来达到，例如进门可以用撬锁、也可以用力量踹开等。随相关动作提升。

## 战斗技能

根据武器类型有不同的战斗技能，战斗技能依靠熟练度提升，普攻也视为一种战斗技能。战斗技能应有CD。视具体技能而定。例如手枪射击为普攻，但几回合就需要一个回合装弹。等级提升提高对应伤害或命中率。例如拳击攻击的伤害，1级为力量+5，2级为力量\*1.2+8，此数值仅为举例。

## 移动与地图

### 大地图移动

玩家位置移动，需要选择移动按钮，画面中部主要界面显示为大地图。地图上显示可移动到的锚点和名称，下方显示对应的地点名称和过去预计时间。（时间与敏捷挂钩）
新地点的解锁依赖于：阅读地图、在旧地点探索（需满足某些条件，或概率）、在某条路径间往返、获得了某些信息等。新地点解锁后，应该有明显的提示。

### 地牢移动

某些区域会有特殊的地点，例如海滩有一个洞窟，玩家可以在在海滩点击洞窟以进入此地牢。地牢中限制保存和快速大地图移动，地牢中移动通过前后左右或者东南西北的交互进行。地牢移动每一步花费少量时间。

## 时间管理与天气

时间会根据玩家动作逐渐流逝。每日的不同时间，不同活动有不同的加成。例如黎明傍晚钓鱼更容易、晚上移动更费时间、晚上睡觉回复更快等。
有季节沦为，每30天一个季节，按照春夏秋冬的顺序依此进行。

## 装备与武器

装备除了提供护甲还提供保暖效果。
装备除了交互获得还可以制作获得。
装备分为头、身、腿、手及饰品，饰品总数限制8个。
武器分近战武器、远程武器，同样可以制作获得。

## 制作

玩家可以进行制作。物品制作需要：1、已知的蓝图（简单物品无需）2、对应的工作台（作为建筑）（简单物品无需） 3、足够的技能和智力 4、消耗时间 5、对应的原材料

## 基地

玩家可以在有限的地点（离谱海滩等平坦地点）建立基地。基地是玩家活动的大本营，在基地可以储存物品、睡觉、制作、建造等。
敌人可能会定期袭击玩家基地。
建造可以建造防御建筑、生产建筑（田地、集雨器）、储存建筑、生活建筑（例如床屋子火堆等）、制作建筑（工作台、铁砧等）等。并不严格分类，例如火堆即可为生活建筑，也是生产烹饪建筑。
基地可以有多个，但是仓储不互通，且长时间不去的基地会因为敌人的破坏而逐渐荒芜。

### 临时基地

玩家可以消耗某些资源就地扎营，不限制地点，除了地牢。临时营地也可以睡觉、储存，但数天就会消失，存的东西也会丢失。

### 埋藏地点

玩家可以在临时基地中花费时间设置埋藏地点，降低消失概率。

# 剧情与任务

主角随着飞机事故，在荒岛的沙滩上醒来，不远处就是飞机的残骸。故事由此开始。
玩家有多个角色可选，不同角色有不同的加成或弱点。
例如：医生：​ 能制作抗生素，工程师：​ 能制作更高级的净水器、修复发电机，猎人：追踪能力更强，剥皮狩猎能力更强，厨师：​ 烹饪的食物更好等。

# 背景

背景基于克苏鲁+SCP的世界观。

在太平洋的中间，有一座无名的岛屿。这座岛本身是正常的热带岛屿，但在五年前，一场海底地震震裂了海沟里的古老沉积层，释放出一种“远古孢子”（The Spores）。孢子并非来自于地球本身，而是早在地球生命诞生之前就降临在地球上的宇宙造物。

孢子核心会无意识地在一定范围内散发能量场，导致生物发生变异。变异后的生物体内会产生新的污染源，即便能量场不存在，接触污染源也会让生物继续变异。

SCP基金会介入后，对孢子本身进行收容，但物质场无法收容。在一段时间的研究后，随着工作人员的陆续变异，最终SCP基金会决定将整个岛一起收容。整个岛屿处于所有物质只进不出的状态。主角因此来到这里。

因此岛上的生物因此变得极具攻击性且外形扭曲，但它们依然是物理存在，可以被长矛刺穿，被子弹打死。

变异的影响分为身体与大脑。身体的变异包括但不限于形态的变化，额外的肢体、超越正常人类的能力。
大脑的变化主要分三种，一种是宗教般的信仰狂热，一种是彻底的疯狂只保留进食冲动，一种是冷漠拒绝理解。

通关结局：
1、修复之前研究基地遗留的岛屿中心最高处的信号塔以呼叫救援。
2、造一个好大船只离开。
保护特定NPC以解锁浪漫结局。
在感染状态离开，解锁0号病人结局。
发现真相，解锁真相结局。
3、低SAN值下，打倒变异人首领，解锁融入结局。
4、发现真相，并通过引爆彻底解决孢子，解锁寂静归零结局。
5、4在自身不死亡条件下，解锁真结局，英雄。
